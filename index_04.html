In this you will get to see about the uses of the Image,Video,Audio And Anchor Tag

Image Tag

The img is used to embed an image in HTML page. Images are not technically inserted into a web page. Images are linked from a web pages. The img tag creates a holding space for refferenced image.

The img has two required attributes: src - Specifies the path to the image alt - Specifies an alternating text for the image, if the image cannot be displayed.

For Example

<img src="https://t4.ftcdn.net/jpg/06/15/21/71/360_F_615217195_y0KQd643vfAm04HEuAjYts2W5UEhRvGx.webp" alt="Nature Img" />

Video Tag

The video tag is used to embed video content in a document, such as a movie clip or other video streams. The video tag contains one or more atribute tags like controls with different video sources. The browser will choose the first source that it will support.

As Example -

<videosrc="Media/Hackathon Project Video (1) (1) (1).mp4" controls></video>

Audio Tag

The audiotag is used to embled sound contain in a document, as music or others audio streams. The <audio > tag contains one or more tags with different audio sources.

For Example:

<audio controls>< source src="Media/Music.mp3" type="audio/mp3"></audio>
